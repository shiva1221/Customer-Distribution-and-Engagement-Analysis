# Customer-Distribution-and-Engagement-Analysis
SQL-based customer engagement and revenue analysis

This repository contains SQL scripts analyzing customer distribution,
engagement behavior, and revenue leakage in an e-commerce dataset.
Each SQL file addresses a specific business question.

## How to Read This Repository
Each SQL file answers a specific business question related to
customer engagement, revenue contribution, or monetization efficiency.
Insights derived from these queries are summarized in the analysis.
